# td-cli
![Npn 版本](https://img.shields.io/badge/td-cli_v0.0.1-green)

## 简介
- 用于快速搭建Web项目命令行工具

## 🚀 模板
- 快速生成 [Vue3-Vite-TypeScript-Pinia 的模板](https://gitee.com/dreamhights/element-plus-pro.git)

## 安装
```
npm install -g td-cli
```

## 使用
### 使用 td-cli 脚手架创建项目
```
td create <projectName>
```

### 更新脚手架到最新版本
```
td update
```

### 查看帮助信息
```
td --help
```

### 查看版本号
```
td -v
td --version
```
